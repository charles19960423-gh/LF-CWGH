# 交互设计 — 资产管理

> **页面路由**：`/assets`  
> **优先级**：P0  
> **版本**：V1.0  
> **包含**：资产管理 + 负债管理（同一页面）

---

## 1. 页面目标

1. 查看和管理所有资产与负债
2. 实时看到净资产数字

## 3. 组件规格

### 3.1 净资产看板（NetWorthBanner）

| 属性 | 规格 |
|------|------|
| 背景 | 主色 `#1e3a5f`，白色文字 |
| 净资产字号 | 32px, bold |
| 总资产/负债 | 16px, 半透明白色 |
| 实时更新 | 增删改资产/负债后立即刷新 |

### 3.2 资产/负债表格（AssetTable / LiabilityTable）

| 属性 | 规格 |
|------|------|
| 表头 | 14px, `slate-500`, uppercase |
| 行高 | 48px |
| 金额 | 等宽字体, 右对齐 |
| 占比 | 仅资产表显示，自动计算 |
| 操作 | 编辑（铅笔）+ 删除（垃圾桶），hover 显示 |
| 合计行 | 底部固定，bold |

## 4. 交互行为

| 动作 | 行为 |
|------|------|
| 点击「+ 新增资产」 | 打开 AssetFormDialog |
| 点击「+ 新增负债」 | 打开 LiabilityFormDialog |
| 点击编辑 ✏️ | 打开 FormDialog，预填数据 |
| 点击删除 🗑 | ConfirmDialog → 确认后删除 |
| 保存 | 关闭弹窗 → 刷新表格 → 更新净资产看板 |
| 输入金额 | 只允许数字，最多 2 位小数 |

## 6. 状态设计

### 6.1 空状态 — 无资产

```
还没有资产记录
添加你的第一项资产，开始构建资产负债表
[ + 新增资产 ]
```

### 6.2 空状态 — 无负债

```
暂无负债记录 🎉
[ + 新增负债 ]
```

### 6.3 删除确认

```
确定删除「招商银行」？
此操作不可撤销。
[ 取消 ]  [ 删除 ]
```

## 7. 验收标准

- [ ] 净资产 = 资产合计 − 负债合计
- [ ] 增删改后 Dashboard 数字同步更新
- [ ] 资产占比之和 = 100%（允许四舍五入误差）
- [ ] 金额不允许负数
- [ ] 删除有确认弹窗
- [ ] 空状态有引导 CTA
