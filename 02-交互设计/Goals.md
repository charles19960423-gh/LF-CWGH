## 3. 组件规格

### 3.1 目标卡片（GoalCard）

| 属性 | 规格 |
|------|------|
| 背景 | 白色，border `slate-200`，圆角 12px |
| 内边距 | 24px |
| 间距 | 卡片间 16px |
| 图标 | emoji 或 lucide，24px |
| 名称 | 18px, font-weight 600 |
| 优先级 | 星星 ⭐ × N，16px |

### 3.2 进度条（GoalProgressBar）

| 属性 | 规格 |
|------|------|
| 高度 | 12px |
| 背景 | `slate-100` |
| 填充 | 主色 `#1e3a5f`；≥ 100% 时变金色 `#c9a227` |
| 百分比 | 进度条右侧，16px, bold |
| 动画 | 首次加载从 0 动画到当前值（500ms ease） |

### 3.3 计算指标行（GoalMetricsRow）

4 列等宽：

| 列 | 数据来源 |
|----|----------|
| 剩余金额 | target − current |
| 剩余时间 | months_between(today, target_date) |
| 每月需存 | ceil(剩余金额 / 剩余月数) |
| 预计完成 | 按当前实际储蓄速度反推（P1）；V1 显示 target_date |

**预计完成逻辑（V1）**：

- 直接显示 `target_date`
- 若 current ≥ target，显示「已达成 🎉」
- 若 remaining_months ≤ 0 且未达成，显示「已逾期 ⚠️」

## 5. 计算逻辑

```typescript
function calcGoalMetrics(goal: Goal) {
  const remaining = goal.target_amount - goal.current_amount;
  const months = monthsBetween(new Date(), goal.target_date);
  const monthlySaving = months > 0 ? Math.ceil(remaining / months) : remaining;
  const progress = (goal.current_amount / goal.target_amount) * 100;

  return { remaining, months, monthlySaving, progress };
}
```

**边界情况**：

| 情况 | 处理 |
|------|------|
| current ≥ target | 进度 100%，显示「已达成 🎉」，每月需存 ¥0 |
| target_date 已过且未达成 | 显示「已逾期 ⚠️」，每月需存 = 剩余金额 |
| months = 0 | 每月需存 = 剩余金额（全部） |

## 6. 状态设计

### 6.1 空状态

```
还没有财务目标
设定你的第一个目标，让 AI 帮你规划路径
[ + 新增目标 ]
```

### 6.2 目标已达成

- 进度条变金色
- 显示「已达成 🎉」徽章
- 每月需存显示 ¥0

### 6.3 目标即将达成（≥ 90%）

- 进度条填充色变为金色
- 预计完成旁显示「即将达成 🎉」
