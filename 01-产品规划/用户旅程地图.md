# 林峰 Life OS — 用户旅程地图

> **版本**：V1.0  
> **日期**：2026-05-31  
> **核心用户**：个体创业者（25-40 岁 · 年入 ¥10 万~100 万）

---

## 1. 五阶段成长旅程

个体创业者使用 Life OS 的完整成长路径：

```
第一阶段          第二阶段          第三阶段          第四阶段          第五阶段
财务混乱    →    开始记录    →    建立目标    →    形成习惯    →    财富增长
                                                    
赚得到钱          录入收支          设100万目标       每周看Dashboard    知道什么业务赚钱
留不住钱          看到净资产        系统算路径        每月复盘           知道钱花在哪里
焦虑              第一次看见        每月需存X         AI建议             增加储蓄率
                  自己有多少
```

---

## 2. 第一阶段：财务混乱

### 用户状态

| 维度 | 描述 |
|------|------|
| 财务 | 赚得到钱，留不住钱 |
| 认知 | 不知道自己真正赚多少钱 |
| 工具 | 微信/支付宝账单 + 脑子 |
| 目标 | 有财务自由梦想，零规划 |
| 情绪 | **焦虑** |

### 典型场景

**张伟（摄影工作室）**：

> 这个月收了 5 万，感觉不错。  
> 但扣完工资、设备、交通、房租、推广… 最后到手可能只有 8 千。  
> 自己从来都没算清楚过。

**李婷（自媒体）**：

> 这个月广告 + 课程 + 咨询赚了 4 万，  
> 下个月可能只有 5 千。  
> 完全不知道哪个业务最赚钱。

### 产品触点（获客）

| 渠道 | 内容 | 行动 |
|------|------|------|
| 林峰公众号 | 「你的净利润是多少？90% 创业者算不对」 | 注册 |
| 短视频 | 「个体创业者必做的 3 步财务规划」 | 注册 |
| 私域社群 | 内测邀请 | 注册 |

### 情绪：😰 焦虑 → 好奇

---

## 3. 第二阶段：开始记录

### 用户行为

```
注册 → 录入资产 → 录入负债 → 记录第一笔收入/支出
```

### 系统响应

- 自动生成 **个人资产负债表**
- 用户 **第一次看见自己的净资产**

### 关键时刻（Aha Moment #1）

```
┌─────────────────────────────────┐
│                                 │
│         你的净资产               │
│         ¥ 57,000                │
│                                 │
│  你之前从来不知道是这个数。         │
│                                 │
└─────────────────────────────────┘
```

### 对应页面

| 动作 | 页面 |
|------|------|
| 录入资产 | `/assets` |
| 录入负债 | `/liabilities` |
| 记第一笔收入 | `/income` |
| 记第一笔支出 | `/expense` |
| 看到净资产 | `/` Dashboard |

### 情绪：😰 焦虑 → 😮 惊喜

### 时长目标：首次激活 ≤ 10 分钟

---

## 4. 第三阶段：建立目标

### 用户行为

```
设定目标：100 万创业基金
  ↓
输入：目标金额 / 当前金额 / 目标日期
  ↓
系统自动生成：
  · 目标金额：¥1,000,000
  · 当前金额：¥57,000
  · 差距：    ¥943,000
  · 剩余时间：36 个月
  · 每月需存：¥26,194
```

### 关键时刻（Aha Moment #2）

```
┌─────────────────────────────────┐
│  🚀 创业基金                     │
│                                 │
│  距离目标：¥943,000              │
│  剩余时间：36 个月               │
│  建议每月存：¥26,194             │
│                                 │
│  原来我需要每月存 2.6 万…          │
│  具体数字让目标变得可执行。        │
│                                 │
└─────────────────────────────────┘
```

### 对应页面

| 动作 | 页面 |
|------|------|
| 创建目标 | `/goals` |
| 查看路径 | `/goals` 或 `/` Dashboard |

### 情绪：😮 惊喜 → 💪 有方向

---

## 5. 第四阶段：形成习惯

### 用户行为

| 频率 | 行为 | 页面 |
|------|------|------|
| 每周 1-2 次 | 打开 Dashboard 看全局 | `/` |
| 每周 2-3 次 | 记录收入/支出 | `/income` `/expense` |
| 每月 1 次 | 更新资产估值 | `/assets` |
| 每月 1 次 | 生成财务复盘 | `/reports` |
| 需要时 | 查看 AI 建议 | `/reports` |

### 系统响应

- Dashboard 实时更新
- 月度复盘自动生成
- AI 给出优化建议

### 典型一月

```
第1周   打开 Dashboard → 看上月结余 → 记本周收入
第2周   记 2-3 笔支出 → 看支出结构
第3周   查看创业基金进度
第4周   生成月度复盘 → 阅读 AI 建议 → 调整下月计划
```

### 情绪：💪 有方向 → 😌 掌控感

## 6. 第五阶段：财富增长

### 用户收获

| 收获 | 具体表现 |
|------|----------|
| 知道什么业务赚钱 | 收入分类统计：项目收入占 60%，最赚钱 |
| 知道什么业务亏钱 | 支出结构：推广费占 35%，ROI 低 |
| 知道钱花在哪里 | 支出结构一目了然 |
| 知道如何增加储蓄率 | AI 建议 + 复盘反馈 |
| 离目标越来越近 | 创业基金 11% → 30% → 65% |

### 情绪：😌 掌控感 → 🚀 进化

### 产品触点

| 功能 | 价值 |
|------|------|
| Dashboard | 一屏看清全局 |
| 收入/支出结构 | 知道赚在哪、花在哪 |
| 目标进度 | 离梦想越来越近 |
| 月度复盘 | 持续优化 |
| 财务复盘 + AI | 决策支持 |

## 8. 关键转化节点

| 节点 | 从 → 到 | 转化率目标 | 失败则 |
|------|---------|-----------|--------|
| T1 | 内容 → 注册 | > 5% | 优化内容钩子 |
| T2 | 注册 → 录入资产 | > 85% | 简化表单 |
| T3 | 录入资产 → 看到净资产 | > 90% | 预填默认值 |
| T4 | 看到净资产 → 设目标 | > 80% | 提供目标模板 |
| T5 | 设目标 → 看到每月需存 | > 95% | 自动计算 |
| T6 | 激活 → 7日回访 | > 50% | 提醒记第一笔 |
| T7 | 7日 → 30日回访 | > 30% | 月度复盘推送 |
| T8 | 30日 → 推荐他人 | > 10% | 分享功能 |

**最关键转化：T3 → T4**（看到净资产 → 设目标）
