## 三、财富系统功能地图

### 3.1 Dashboard 财务驾驶舱

用户打开 App 第一眼看到的内容：

**财富概览**

| 指标 | 公式 |
|------|------|
| 总资产 | SUM(assets.amount) |
| 总负债 | SUM(liabilities.amount) |
| 净资产 | 总资产 − 总负债 |

**本月概览**

| 指标 | 公式 |
|------|------|
| 本月收入 | SUM(transactions WHERE type=income) |
| 本月支出 | SUM(transactions WHERE type=expense) |
| 本月结余 | 本月收入 − 本月支出 |
| 储蓄率 | 结余 ÷ 收入 × 100% |

**财富趋势**（P1 · 图表）

- 收入趋势（近 6/12 月）
- 支出趋势（近 6/12 月）
- 净资产趋势（近 6/12 月）

**目标进度**

- 创业基金 / 应急基金 / 买房基金 …
- 完成度 + 剩余金额 + 每月需存

### 3.2 收入管理

**新增收入字段**：

| 字段 | 类型 | 必填 |
|------|------|------|
| date | 日期 | ✓ |
| amount | 金额 | ✓ |
| category | 分类 | ✓ |
| account | 账户 | |
| note | 备注 | |

**收入分类**（为创业者设计）：

```
工资 | 项目收入 | 咨询收入 | 课程收入 | 广告收入 | 投资收益 | 其他
```

---

### 3.3 支出管理

**新增支出字段**：同收入结构

**支出分类**（为创业者设计，含商业成本）：

```
餐饮 | 住房 | 交通 | 娱乐 | 学习 | 健康
商业投入 | 营销推广 | 员工工资 | 设备采购 | 其他
```

> 关键差异：必须有 **商业投入 · 营销推广 · 员工工资 · 设备采购** — 这是创业者专属分类，记账 App 没有。

---

### 3.4 资产管理

**资产类型**：

```
现金 | 银行卡 | 支付宝 | 微信 | 股票 | 基金 | 房产 | 车辆 | 公司股权
```

**字段**：名称、类型、金额

---

### 3.5 负债管理

**负债类型**：

```
信用卡 | 房贷 | 车贷 | 经营贷 | 朋友借款
```

**字段**：名称、类型、金额、利率

---

### 3.6 财务目标

**目标模板**：

```
应急基金 | 创业基金 | 买房基金 | 教育基金 | 退休基金
```

**字段**：目标名称、目标金额、当前金额、目标日期

**自动计算**：

```
完成度     = 当前金额 ÷ 目标金额 × 100%
缺口       = 目标金额 − 当前金额
每月需存   = 缺口 ÷ 剩余月份
```

---

### 3.7 财务复盘

**自动生成内容**：

**本月收入排行**

```
咨询收入   40%
项目收入   35%
课程收入   25%
```

**本月支出排行**

```
商业投入   35%
住房       20%
餐饮       15%
```

**AI 建议示例**

```
· 本月商业投入增长 50%，建议跟踪 ROI
· 学习支出占比 10%，属于合理区间
· 储蓄率 42%，高于建议的 30%，状态健康
```

---

### 3.8 AI 财务顾问

集成在 `/reports` 复盘页 + Dashboard 智能卡片（V1 不单独设页）。

**能力**：
