# 林峰 Life OS — 用户画像 V1

> **版本**：V1.0  
> **日期**：2026-05-31  
> **作者**：林峰  
> **核心锁定**：个体创业者 · 25-40 岁 · 年入 ¥10 万~100 万

---

## 0. 核心结论

**V1 只服务一类人：个体创业者。**

不是普通人，不是大厂白领，不是学生。

这是林峰 IP 最自然、最懂、最容易做内容、最容易变现的一群人。

```
林峰 IP 资产              用户重叠
─────────────            ─────────
老板 IP           ←→     个体创业者 / 小老板
企业增长          ←→     有业务但缺系统的人
林峰系统论        ←→     相信系统 > 意志力
个人认知系统      ←→     知道很多、执行很少
内容创业          ←→     自媒体 / 培训 / 顾问
```

---

## 1. 核心用户定义

| 维度 | 定义 |
|------|------|
| 身份 | 个体创业者 |
| 年龄 | 25 ~ 40 岁 |
| 年收入 | ¥100,000 ~ ¥1,000,000 |
| 职业 | 摄影师、剪辑师、设计师、顾问、自媒体博主、培训老师、AI 创业者 |
| 收入特征 | 不稳定、多来源、难预测 |
| 财务状态 | 没有财务系统，不知道自己真正赚多少钱 |
| 心理特征 | 有财务自由梦想，但缺乏量化路径 |
| 方法论认同 | 相信「系统管理 > 意志力管理」 |

## 2. 用户画像 01 — 摄影工作室老板

```
┌──────────────────────────────────────────┐
│  张伟 · 32 岁 · 摄影工作室老板              │
│  年入 ¥300,000 · 团队 2 人                │
└──────────────────────────────────────────┘
```

### 基本信息

| 项 | 内容 |
|----|------|
| 姓名 | 张伟 |
| 年龄 | 32 岁 |
| 城市 | 二线城市 |
| 业务 | 婚礼摄影 + 商业拍摄 |
| 团队 | 2 人（自己 + 1 名摄影师） |
| 年营收 | ¥300,000 ~ ¥500,000 |
| 个人年入 | ~¥300,000 |

### 获客渠道

- 朋友圈口碑
- 老客户转介绍
- 抖音/小红书短视频

### 财务状况（他自己算不清的）

| 项 | 金额 | 备注 |
|----|------|------|
| 本月收款 | ¥50,000 | 看起来很多 |
| 摄影师工资 | −¥8,000 | |
| 设备折旧 | −¥3,000 | 相机/镜头/灯光 |
| 交通费 | −¥2,000 | 各地跑片 |
| 房租 | −¥4,500 | 工作室 |
| 推广费 | −¥5,000 | 抖音投放 |
| 其他 | −¥19,500 | 餐饮/杂费/耗材 |
| **实际净利润** | **~¥8,000** | **自己都不知道** |

### 痛点

| # | 痛点 | 原话 |
|---|------|------|
| 1 | **不知道自己真正赚多少钱** | 「这个月收了 5 万，但到手多少？」 |
| 2 | **收款 ≠ 利润** | 「表面看收入不错，扣完美成本所剩无几」 |
| 3 | **没有长期规划** | 「三年想存 100 万，但完全没路径」 |
| 4 | **个人与公司混在一起** | 「工作室的钱和个人零花钱不分」 |
| 5 | **忙到没空记账** | 「知道该记，就是坚持不了」 |

### 目标

- 三年存 ¥1,000,000（买房 + 扩大工作室）
- 目前没有规划，没有路径

### 对 Life OS 的期望

> 「告诉我：扣除所有成本后，我真正赚了多少钱，离 100 万还有多远，每月该存多少。」

### 使用场景

| 场景 | 行为 |
|------|------|
| 收到客户定金 ¥20,000 | 记一笔项目收入 |
| 发工资 | 记一笔商业投入/支出 |
| 月底 | 打开 Dashboard 看本月净利润 |
| 季度 | 看创业基金进度 |

## 3. 用户画像 02 — 自媒体创业者

```
┌──────────────────────────────────────────┐
│  李婷 · 28 岁 · 自媒体创业者               │
│  年入 ~¥400,000 · 一人公司                 │
└──────────────────────────────────────────┘
```

### 基本信息

| 项 | 内容 |
|----|------|
| 年龄 | 28 岁 |
| 城市 | 一线城市（远程） |
| 业务 | 知识博主 + 线上课程 + 咨询 |
| 团队 | 1 人 |
| 粉丝 | 抖音 15 万 + 公众号 3 万 |

### 收入来源（每月都不一样）

| 来源 | 占比（波动） | 月收范围 |
|------|-------------|----------|
| 广告/商单 | 30~50% | ¥5,000 ~ ¥30,000 |
| 课程销售 | 25~40% | ¥3,000 ~ ¥50,000 |
| 咨询 | 15~25% | ¥0 ~ ¥20,000 |
| 带货 | 5~15% | ¥0 ~ ¥10,000 |

### 痛点

| # | 痛点 | 原话 |
|---|------|------|
| 1 | **收入极不稳定** | 「这个月 5 万，下个月可能 5 千」 |
| 2 | **不知道哪种业务最赚钱** | 「课程卖得多但咨询单价高，哪个更好？」 |
| 3 | **没有利润概念** | 「赚了钱但账户里没剩多少」 |
| 4 | **缺乏分析工具** | 「想看收入结构，Excel 太麻烦」 |

### 需求

| 需求 | 对应功能 |
|------|----------|
| 分析收入结构 | 收入管理 + 分类占比 |
| 分析利润结构 | 支出结构 + 结余 |
| 分析现金流 | Dashboard 本月收支 |
| 知道哪种业务最赚钱 | 按来源/分类统计（V2 深化） |

### 对 Life OS 的期望

> 「帮我看清：哪个业务最赚钱，钱从哪来、到哪去，淡季能撑多久。」

## 4. 用户画像 03 — 小老板

```
┌──────────────────────────────────────────┐
│  王磊 · 35 岁 · 小型企业老板               │
│  企业营收 ¥100-500 万 · 团队 8 人          │
└──────────────────────────────────────────┘
```

### 基本信息

| 项 | 内容 |
|----|------|
| 年龄 | 35 岁 |
| 城市 | 二线城市 |
| 业务 | 电商 + 私域运营 |
| 团队 | 8 人 |
| 企业年营收 | ¥1,000,000 ~ ¥5,000,000 |
| 个人年入 | ~¥800,000 |

### 财务状况

| 账户 | 状态 |
|------|------|
| 个人账户 | 有钱（¥200,000+） |
| 企业账户 | 有钱（¥500,000+） |
| 个人资产 | 房产 ¥300 万、车 ¥30 万 |
| 个人负债 | 房贷 ¥150 万、经营贷 ¥50 万 |

### 痛点

| # | 痛点 | 原话 |
|---|------|------|
| 1 | **个人与企业资产混在一起** | 「个人账户有钱，企业账户也有钱，但我自己有多少？」 |
| 2 | **永远不知道净资产是多少** | 「房产、股票、公司股权、存款… 从没算过总数」 |
| 3 | **企业赚钱但个人没感觉** | 「公司年利润 100 万，我个人能花多少？」 |
| 4 | **缺少长期规划** | 「想 5 年财务自由，但没有任何量化计划」 |

### 对 Life OS 的期望

> 「帮我算清楚：抛开公司，我个人到底有多少净资产，离目标还有多远。」

> V1 不做企业财务，仅记录 **个人层面** 的资产/负债/目标。

## 5. 三类用户共性

| 共性 | 说明 |
|------|------|
| 赚得到钱 | 年入 10 万~100 万，有赚钱能力 |
| 留不住钱 / 算不清 | 没有财务系统，不知道真实利润 |
| 有梦想无路径 | 想存 100 万/买房/自由，但没有计划 |
| 认同系统论 | 林峰 IP 受众，相信系统 > 意志力 |
| 愿意手动录入 | V1 无自动同步，接受手动（如果价值够大） |

---

## 6. 共性痛点 → 产品价值

```
痛点                          Life OS 价值
─────────────────────────────────────────────
不知道自己真正赚多少钱    →    Dashboard 一屏看清
收了钱但利润很少         →    支出结构分析
没有长期规划             →    财务目标 + 每月需存
个人/企业混在一起        →    个人资产负债表
不知道哪种业务最赚钱      →    收入分类统计（V2 深化）
坚持不了记账             →    不是记账，是驾驶舱（降低频率）
```

## 7. 非目标用户

| 排除 | 原因 |
|------|------|
| 普通上班族 | 固定工资，银行 App 够用 |
| 学生 | 无收入 |
| 年入 100 万+ 大老板 | 有 CFO，需求超出 V1 |
| 纯记账用户 | 「今天花了 35 块」→ 随手记已满足 |
